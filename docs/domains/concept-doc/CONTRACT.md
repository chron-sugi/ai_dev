# Contract — concept-docs

Current-state registry of this domain's contract. Each entry cites the ADR
that granted it; this file is never the write path for a new grant.

## Layout

- Each domain is documented in `docs/domains/<domain>/`, containing exactly
  `CONCEPT.md` and `CONTRACT.md` — [ADR-0009](../../adrs/adr-0009-per-domain-docs-folder-concept-and-contract.md)
- `docs/concepts/` holds cross-cutting concepts only — [ADR-0009](../../adrs/adr-0009-per-domain-docs-folder-concept-and-contract.md)
